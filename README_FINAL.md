# نظام إدارة المراجعة الداخلية الشامل (Khatwa System)

نظام متطور مصمم خصيصاً لإدارة عمليات المراجعة الداخلية وفقاً لأعلى المعايير الدولية.

## المميزات الرئيسية:
- **نظام شامل ومتكامل:** يغطي كافة مراحل المراجعة من التخطيط إلى التقارير.
- **التوافق الدولي:** مصمم وفقاً لمعايير (IIA, COSO, ISO).
- **واجهة احترافية:** تصميم عربي (RTL) بالكامل ومتجاوب مع جميع الأجهزة.
- **التحليل الذكي:** أدوات متقدمة للتحليل المالي وتقييم المخاطر.
- **التقارير:** نظام توليد تقارير احترافية جاهزة للطباعة.

## متطلبات التشغيل:
- Node.js (v18+)
- npm أو pnpm

## طريقة التشغيل:
1. فك ضغط الملف.
2. افتح المجلد في الطرفية (Terminal).
3. قم بتثبيت التبعيات: `npm install`
4. ابدأ التشغيل في وضع التطوير: `npm run dev`
5. للإنتاج: `npm run build` ثم `npm run start`

تم التطوير بواسطة فريق Khatwa System.
